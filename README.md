# AMABAL Masala 🌶️

Traditional South Indian Spice Blends from Erode, Tamil Nadu.

## Products
- Murungai Podi (முருங்கை பொடி)
- Puli Podi (புளி பொடி)
- Ellu Podi (எள்ளு பொடி)
- Curry Leaf Podi (கறிவேப்பிலை பொடி)

## Tech
- Pure HTML/CSS/JS (single file)
- Fully responsive (mobile, tablet, desktop)
- Hosted on Vercel
